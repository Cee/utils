Spell Checker Tool Reference

Requirement

Python 3 and above

Usage

/utils/html_spell.py <html file to check> <Original Dictionary> <Custom Dictionary>

Current Features

    Matches words with dictionary (/utils/English.txt).
    Provision to add new words to dictionary or ignore.
    Uses Regular Expression to skip words containing any characters other than a letter or hyphen.
        Adds new words to a custom dictionary rather than the original one.
