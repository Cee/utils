Spell Checker Tool Reference

Requirement

Python 3 and above

Usage

html_spell.py <html file> <main dict> <custom dict> [-e] [-h]

Current Features

    Matches words with dictionary (/utils/English.txt).
    Provision to add new words to dictionary or ignore.
    Uses Regular Expression to skip words containing any characters other than a letter, hyphen or apostrophe.
    Adds new words to a custom dictionary rather than the original one.
    Exit with error code can be enabled with -e cmd line argument.
