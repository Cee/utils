Creating Embedded Python Consoles 

Here are the steps for embedding a Python console (from PythonAnywhere, at present) inside a web page for one of our courses, or otherwise. 

  1. Create a gist on GitHub by copying for repos, from the gist get https hash url.
  2. modify ptml file, include python_anywhere.txt. This file exist algorithms/templates directory.
  3. Include instruction for how to load and run the code in python console. Use example from textbook.
  4. Test, test, test
